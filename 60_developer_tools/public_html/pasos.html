    Node y NPM

  60. ver como funciona en tiempo real un proyecto -> lite-server - NodeJS

    61. https://github.com/johnpapa/lite-server

    • primero se instala el lite-server
    • npm init
    • npm i -g npm para actualizar npm
    • instalamos el lite-server
    • npm install lite-server --save-dev
    • dentro del fichero package.json, dentro de "scripts" se añade
    • "dev": "lite-server"
    • y por ultimo se lanza el servidor
    • npm run dev
